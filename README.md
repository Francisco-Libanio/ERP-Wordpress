# Gestor de produtos

## Objetivo

O objetivo desse repositório é a construção de um gerenciador de produtos para e-comerces.
O primiero passo será construir as páginas basicas so sistema e criar as rotas entre elas.

## Tecnologias

- Flask

# Páginas

- Homepage
- Seleção de módulos
- Cadastro de produtos
- Estoque
- Perfil do usuário
